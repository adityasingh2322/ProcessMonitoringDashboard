# 🖥️ Real-Time Process Monitoring Dashboard

A **desktop-based system monitoring tool** built using **Python, Tkinter, and Matplotlib** that provides real-time insights into system processes, CPU usage, and memory utilization.

---

## 🚀 Features

- 📊 **Real-Time Process Monitoring**
  - View running processes with:
    - PID
    - Name
    - User
    - CPU usage
    - Memory usage
    - Priority level

- 🔍 **Process Filtering**
  - Filter processes by name for quick access

- ❌ **Kill Process**
  - Terminate selected processes directly from the dashboard

- ⚙️ **Change Process Priority**
  - Increase or decrease priority of running processes

- 📈 **Live CPU & Memory Graphs**
  - Visual representation of system performance

- ⚠️ **Alert System**
  - Get warnings when:
    - CPU usage > 80%
    - Memory usage > 80%

---

## 🛠️ Tech Stack

- **Python**
- **Tkinter** (GUI)
- **psutil** (System monitoring)
- **Matplotlib** (Graph visualization)
- **Threading**
